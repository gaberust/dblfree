What is DBL Free?

DBL Free is a reference to a double free memory error, which is both a bug and software vulnerability. DBL Free hosts both Gabe Rust's blog and portfolio as he pursues a career in cyber security. DBL Free content will include many development and security topics, such as general development, web application security, and CTF write-ups.

About the Author

Gabe is a self-taught developer who took an interest in programming at age 16, whose interest quickly became a passion. As he grew more familiar with web development in particular, web application security and safely handling untrusted user input caught his attention. He has been studying development since 2015 and information security since 2018. Wanting to write secure applications got him thinking like a hacker. What would I do to break this constantly ran across his mind as he was writing code and he soon realized that he enjoys security just as much- if not more- than development. Gabe Rust is currently pursuing enjoying a career as a security consultant.
